## ⌨ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl` + `K` | 打开全局搜索 |
| `/` | 快速聚焦搜索框 |
| `Ctrl` + `B` | 切换侧边栏显示/隐藏 |
| `Ctrl` + `D` | 切换暗色模式（深色主题） |
| `Ctrl` + `↑` | 回到页面顶部 |
| `Esc` | 关闭当前弹窗（搜索/面板） |
| `?` | 显示快捷键帮助面板 |
| `↑` `↓` | 在搜索结果中上下导航 |
| `Enter` | 跳转到选中的搜索结果 |

## 🎨 功能特性

- **🌓 暗色模式** — 自适应 GitHub 风格暗色主题，代码块、表格、引用全面适配
- **📊 阅读进度条** — 页面顶部绿色进度条实时反映阅读进度
- **📑 本页目录** — 自动提取标题生成浮动导航，点击跳转
- **🔍 全文搜索** — 页面内实时搜索（高亮 + 结果列表 + 键盘导航）
- **💾 阅读位置记忆** — 刷新或切换页面后自动恢复到上次阅读位置
- **🖼 图片灯箱** — 点击文档内图片全屏预览
- **📝 代码行号** — 代码块左侧显示行号，便于引用
- **🔗 标题锚点** — 悬停标题出现 # 链接图标，点击复制 URL
- **📂 手风琴侧边栏** — 文件夹单击展开/折叠，自动定位当前页面
- **⬆ 回到顶部** — 右下角按钮，含环形滚动进度条
