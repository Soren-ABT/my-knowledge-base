## 为什么需要一个知识库？

在研究生阶段，我意识到知识的积累速度远超遗忘速度——但前提是你有一个系统来承载它们。

我曾经用过很多工具：Notion、Obsidian、纸质笔记、GitHub Issues……但它们要么太封闭，要么太零散。最终我决定**自己搭建**一个公开的知识库：

- **公开驱动责任感** — 当你写的笔记有读者时（哪怕只有未来的自己），你会更认真
- **Markdown 可迁移** — 纯文本格式，20 年后仍然可以打开
- **数学公式支持** — LaTeX 实时渲染，适合数学/物理笔记
- **代码高亮** — 适合记录算法实现和技术方案

## 这个知识库里有什么？

目前主要覆盖以下几个方向：

| 方向 | 内容 |
|------|------|
| 复分析 | 复变函数、积分变换的复习笔记 |
| 普通物理 | 力学部分的系统梳理 |
| 数学建模 | 论文写作方法论 |
| 深度学习 | AMC 方向的研究记录 |

## 技术栈

本站基于 Astro 6 构建，核心技术：

- **Astro 6** + **Tailwind CSS 4** — 静态生成，零 JS 首屏
- **Three.js** — 首页 3D 交互场景
- **MathJax** — 服务端 LaTeX 渲染
- **Shiki** — 代码语法高亮
- **GitHub Pages / Vercel** — 部署
